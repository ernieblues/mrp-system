# MRP System

Material Requirements Planning (MRP) system built with ASP.NET Core, React TypeScript, and SQL Server for managing inventory, purchasing, and production workflows.

## Tech Stack

- React TypeScript
- ASP.NET Core MVC
- ASP.NET Core Identity
- .NET 10
- SQL Server

## Features

- Inventory and material tracking
- Purchase order management
- Role-based authentication (Admin, Manager, User)

## Getting Started

### Prerequisites

- .NET SDK (version 10.0 or later)
- Node.js
- SQL Server Developer Edition

### Configuration

- Rename `appsettings.Development.json.example` to `appsettings.Development.json`
- Update the connection string to your SQL Server instance

### Run

- Set the Server project as the startup project
- Run the application
- Navigate to the URL shown in the output window
